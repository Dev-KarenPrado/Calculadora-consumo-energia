> 💡 **Dica:** Economizar energia começa com o conhecimento do consumo!

Projeto desenvolvido para estimar o consumo mensal de aparelhos eletrodomésticos e calcular o custo estimado em Reais.

## 🚀 Funcionalidades
- Cálculo de consumo em **kWh**.
- Estimativa de custo mensal (baseado na tarifa de R$ 0,75/kWh).
- Interface simples e intuitiva via terminal.

## 📖 Como usar
1. Execute o arquivo `app.py`.
2. Informe o nome do aparelho, a potência (Watts) e as horas de uso diário.
3. Visualize o resultado formatado no ecrã.

## 🧮 Fórmula utilizada
$ConsumoMensal = (Potência \times Horas \times 30) / 1000$

## 📌 Exemplo de uso

Aparelho: Chuveiro  
Potência: 5500W  
Uso diário: 1 hora  
Valor do kWh: R$ 0,90  

🔎 Resultado:
Consumo mensal: 165 kWh  
Custo estimado: R$ 148,50
